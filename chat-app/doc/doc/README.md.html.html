<p>Requirements for project:</p>
<ul>
<li>Express back end with get/post endpoints</li>
<li>Mongodb for storage</li>
<li>Websocket</li>
<li>React + Redux with routes</li>
<li>Microserver architecture, gateway must only server to route requests, break up microservices by topic, ie user services, comment servicesâ€¦ can be done with express or NGNIX</li>
<li>redis for shared data</li>
<li>Backend components must be dockerized and running in Docker swarm mode</li>
<li>Node components can use either docker or pm2 (cluster)</li>
<li>Use at least 1 kafka message topic to broadcast real time updates (can get extra credit with this)</li>
<li>Bonus points for an electron port with some native functionality.</li>
<li>Finally reverse proxy with NGNIX</li>
</ul>
